# 오늘의 메뉴는?

## 프로젝트 소개
식사 선택을 즐겁고 간편하게 도와주는 앱 입니다.  
카테고리별 다양한 음식 추천과 랜덤 메뉴 선택 기능및 즐겨찾기와 커스텀 음식 목록 기능으로 편리하게 그날의 메뉴를 정할 수 있습니다.
